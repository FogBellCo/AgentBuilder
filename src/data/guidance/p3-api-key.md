# Working with Confidential Data (P3)

## What this means

Your data is confidential and requires authorization beyond standard SSO. This typically means you need an API key, VPN access, or explicit approval from a data steward.

## Getting access

1. **Identify the data steward** — Every confidential dataset has an assigned steward who controls access
2. **Request API credentials** — Submit a request through your department's IT liaison
3. **Set up secure access** — You may need to install VPN software or configure API keys
4. **Get approval** — Wait for the data steward to approve your use case

## Working with AI tools

- Use only UCSD-approved AI platforms that support API key authentication
- Never copy confidential data into public AI tools (like public ChatGPT)
- Maintain an audit trail of how the data is accessed and processed
- Consider de-identifying data before AI processing when possible

## Need help?

- [UCSD Information Security](https://blink.ucsd.edu/technology/security/) — For security questions
